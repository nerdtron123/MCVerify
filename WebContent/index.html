For Developers

Easily verify Minecraft Account ownership.

make a request like this one:
verify.mcverify.ga/auth?redirect_uri=YOUR_REDIRECT_URI&scope=test&client_id=secret


We will return the user to YOUR_REDIRECT_URI with the following attached: ?code=RESPONSE_UUID
You can then make a request to verify.mcverify.ga/poll?token=RESPONSE_UUID this will return the minecraft username of the user
